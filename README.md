# GDK Site (WIP)
The official website codebase for GameDev Knights (GDK) at the University of Central Florida (UCF). 

This site is built with basic HTML/CSS/JS to avoid the overhead of a overly complex framework. 

`js/components.js` defines reusable code blocks like the header, footer, dropdowns, etc.

Online payment (purchasing memberships) is handled via Stripe payment links.

Contact Aden Thompson (discord: algaebucket) if you have any questions.

## Adding events to the homepage

1. Login to UCF Events system and Create a New Event: https://events.ucf.edu/help/#sign-up-log-in
2. Under Calendar, select GameDev Knights.
3. Fill out the rest of the details and create the event.
4. The website will automatically update with the new event on the homepage.
